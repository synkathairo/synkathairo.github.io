Matrix exponential

Definition

eAt≜I+At+A2t22!+...=∑i=0∞Aitii!deAtdt=A[I+At+A2t22!+...]=AeAt\begin{eqnarray} e^{At} \triangleq I + At + A^2\frac{t^2}{2!} + ... = \sum_{i=0}^\infty A^i \frac{t^i}{i!} \\ \frac{de^{At}}{dt} = A [I + At + A^2 \frac{t^2}{2!} + ...] = Ae^{At} \end{eqnarray}

Properties

  • eA0=I+A0+A2022!+...=Ie^{A0} = I + A0 + A^2 \frac{0^2}{2!} + ... = I
  • (eAt)−1=e−At(e^{At})^{-1} = e^{-At}
  • ddteAt=AeAt\frac{d}{dt} e^{At} = A e^{At}

Notes

If AA is 1×11 \times 1 i.e. scalar, this becomes the power series, eat=∑k=0∞(at)kk!=1+at+a2t22!+...e^{at} = \sum_{k=0}^\infty \frac{(at)^k}{k!} = 1 + at + a^2\frac{t^2}{2!} + ... which matches a common definition of the exponential function
