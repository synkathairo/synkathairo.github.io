convex set

Definition

A set 𝒮\mathcal{S} is convex if for any 𝐱,𝐲∈𝒮\mathbf{x},\mathbf{y} \in \mathcal{S}, λ∈[0,1]\lambda \in [0,1]: (1−λ)𝐱+λ𝐲∈𝒮(1-\lambda)\mathbf{x} + \lambda \mathbf{y} \in \mathcal{S}

Theorem:

Let f:S→ℝf : S \longrightarrow \mathbb{R} be a function defined on the convex subset SS of a real linear space LL. Then, ff is convex on SS if and only if its epigraph is a convex subset of S×ℝS × \mathbb{R}; ff is concave if and only if its hypograph is a convex subset of S×ℝS × \mathbb{R}.
