theorem, security levels of matrix game players

Theorem

In every Matrix game A={aij}A = \{a_{ij}\},

  1. the security level of each player is unique
  2. there exists at least one security strategy for each player
  3. The security level of P1 (the minimizer) never falls below the security level of P2 (the maximizer), i.e. miniaij*=V_(A)≤V¯(A)=maxjai*j\min_i a_{ij^*} = \underline{V}(A) \leq \overline{V}(A) = \max_j a_{i^* j} where i*i^* and j*j^* denote security strategies for P1 and P2 respectively

(note: P1 as minimizer, P2 as maximizer is a convention which is reversed in some textbooks)
