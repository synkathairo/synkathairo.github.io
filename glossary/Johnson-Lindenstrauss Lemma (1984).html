Johnson-Lindenstrauss Lemma (1984)

Lemma

For any set of nn data points 𝐪1,...,𝐪n∈ℝd\mathbf{q}_1,...,\mathbf{q}_n \in \mathbb{R}^d, there exists a linear map 𝚷:ℝd→ℝk\mathbf{\Pi}: \mathbb{R}^d \rightarrow \mathbb{R}^k where k=O(lognϵ2)k=O(\frac{\log{n}}{\epsilon^2}) such that for all i,ji,j, (1−ϵ)||𝐪i−𝐪j||2≤||𝚷𝐪i−𝚷𝐪j||2≤(1+ϵ)||𝐪i−𝐪j||2(1-\epsilon)||\mathbf{q}_i-\mathbf{q}_j||_2 \leq ||\mathbf{\Pi q}_i-\mathbf{\Pi q}_j||_2 \leq (1+\epsilon)||\mathbf{q}_i-\mathbf{q}_j||_2

Equivalently, (1−ϵ)||𝐪i−𝐪j||22≤||𝚷𝐪i−𝚷𝐪j||22≤(1+ϵ)||𝐪i−𝐪j||22(1-\epsilon)||\mathbf{q}_i-\mathbf{q}_j||_2^2 \leq ||\mathbf{\Pi q}_i-\mathbf{\Pi q}_j||_2^2 \leq (1+\epsilon)||\mathbf{q}_i-\mathbf{q}_j||_2^2 because for small ϵ\epsilon, (1+ϵ)2=1+O(ϵ)(1+\epsilon)^2 = 1+O(\epsilon) and (1−ϵ)2=1−O(ϵ)(1-\epsilon)^2 = 1-O(\epsilon).

Equivalently, (1−ϵ)||𝚷𝐪i−𝚷𝐪j||22≤||𝐪i−𝐪j||22≤(1+ϵ)||𝚷𝐪i−𝚷𝐪j||22(1-\epsilon)||\mathbf{\Pi q}_i-\mathbf{\Pi q}_j||_2^2 \leq ||\mathbf{q}_i-\mathbf{q}_j||_2^2 \leq (1+\epsilon)||\mathbf{\Pi q}_i-\mathbf{\Pi q}_j||_2^2 because for small ϵ\epsilon, 11+ϵ=1−O(ϵ)\frac{1}{1+\epsilon}=1-O(\epsilon) and 11−ϵ=1+O(ϵ)\frac{1}{1-\epsilon}=1+O(\epsilon).

Proof using Distributional Johnson-Lindenstrauss Lemma:

We have a set of vectors 𝐪1,...,𝐪n\mathbf{q}_1,...,\mathbf{q}_n. Fix i,j∈{1,...,n}i,j \in \{1,...,n\}.

Let 𝐱=𝐪i−𝐪j\mathbf{x}=\mathbf{q}_i - \mathbf{q}_j. By linearity, 𝚷𝐱=𝚷(𝐪i−𝐪j)=𝚷𝐪i−𝚷𝐪j\mathbf{Πx} = \mathbf{Π}(\mathbf{q}_i −\mathbf{q}_j) = \mathbf{Πq}_i −\mathbf{Πq}_j.

By distributional JL Lemma, with probability 1−δ1-\delta, (1−ϵ)||𝐪𝐢−𝐪𝐣||2≤||𝚷𝐪i−𝚷𝐪j||2≤(1+ϵ)||𝐪𝐢−𝐪𝐣||2(1-\epsilon)||\mathbf{\mathbf{q}_i - \mathbf{q}_j}||_2 \leq ||\mathbf{\Pi q}_i - \mathbf{\Pi q}_j||_2 \leq (1+\epsilon)||\mathbf{\mathbf{q}_i - \mathbf{q}_j}||_2

Finally, set δ=1n2δ = \frac{1}{n^2} . Since there are <n2< n^2 total i,ji,j pairs, by a Union Bound we have that with probability 9/109/10, the above will hold for all i,ji, j, as long as we compress to:

k=O(log(1/(1/n2))ϵ2)=O(lognϵ2)k=O(\frac{\log(1/(1/n^2))}{\epsilon^2})=O(\frac{\log{n}}{\epsilon^2}) dimensions.

This is a type of Euclidean Dimensionality Reduction.
