approachable set

Definition

Let 𝒞\mathscr{C} be a convex and closed subset of ℝL\mathbb{R}^L. The set 𝒞\mathscr{C} is approachable by decision-maker ii if there is a procedure (behavior strategy) for ii that guarantees that the average vector payoff DtD_t approaches the set 𝒞\mathscr{C} (i.e. dist(Dt,𝒞)→0\operatorname{dist}(D_t,\mathscr{C}) \to 0 almost surely as t→∞t \to \infty), regardless of the choices of the opponent −i-i.
