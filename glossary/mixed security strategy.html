mixed security strategy

Definition

Vector y*∈Yy^* \in Y is called a mixed security strategy for P1, in matrix game AA, if following inequality holds for all y∈Yy \in Y: V¯m(A)≜maxz∈Zy*′Az≤maxz∈Zy′Az,y∈Y\overline{V}_m(A) \triangleq \max_{z \in Z} {y^*}' Az \leq \max_{z \in Z} y' Az, \quad y \in Y

(note convention: P1 minimizer, P2 maximizer)
