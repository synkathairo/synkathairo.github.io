Strict preference relation

Definition

Given the preference relation ≿i\succsim_i we can define the corresponding strict preference relation ≻i\succ_i, which describes when player ii strictly prefers one outcome to another: x≻iy⇔x≿iy and y≿̸ixx \succ_i y \iff x \succsim_i y \text{ and } y \not\succsim_i x
